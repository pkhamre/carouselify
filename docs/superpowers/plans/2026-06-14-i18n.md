# i18n Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add internationalization for English, Norwegian Bokmål, and Polish using next-intl with subpath routing

**Architecture:** next-intl v3 with App Router. Locale prefix on all routes (`/en/`, `/nb/`, `/pl/`). Middleware detects locale from cookie → Accept-Language → `en`. Messages in `messages/{locale}.json`. All 13 components with user-facing strings use `useTranslations('namespace')`.

**Tech Stack:** next-intl v3, Next.js 15 App Router

```tsx
import { redirect } from "next/navigation";
import { cookies } from "next/headers";
import { match } from "@formatjs/intl-localematcher";
import Negotiator from "negotiator";

const locales = ["en", "nb", "pl"];
const defaultLocale = "en";

function getLocale(request: Request): string {
  const headers = { "accept-language": request.headers.get("accept-language") || "" };
  const languages = new Negotiator({ headers }).languages();
  return match(languages, locales, defaultLocale);
}

export default async function SharedRedirect({
  params,
}: {
  params: { id: string };
}) {
  const cookieStore = await cookies();
  const localeCookie = cookieStore.get("NEXT_LOCALE")?.value;
  
  let locale = localeCookie || defaultLocale;
  if (!locales.includes(locale)) locale = defaultLocale;
  
  redirect(`/${locale}/s/${params.id}`);
}
```

Note: This file needs `negotiator` and `@formatjs/intl-localematcher` — either add these deps or simplify to always redirect to `/en/...`.

Simpler version that always redirects to `en`:
```tsx
import { redirect } from "next/navigation";

export default async function SharedRedirect({
  params,
}: {
  params: { id: string };
}) {
  redirect(`/en/s/${params.id}`);
}
```

### Task 19: Create LocaleSwitcher component

**Files:**
- Create: `frontend/src/components/LocaleSwitcher.tsx`

- [ ] **Step 1: Create the component**

```tsx
"use client";

import { useLocale } from "next-intl";
import { usePathname, useRouter } from "next-intl/client";

export function LocaleSwitcher() {
  const locale = useLocale();
  const pathname = usePathname();
  const router = useRouter();

  const switchTo = (nextLocale: string) => {
    router.replace(`/${nextLocale}${pathname}`);
  };

  return (
    <select
      value={locale}
      onChange={(e) => switchTo(e.target.value)}
      className="text-sm border border-gray-300 dark:border-gray-600 rounded-md px-2 py-1 bg-white dark:bg-gray-800 text-gray-900 dark:text-gray-100"
      aria-label="Select language"
    >
      <option value="en">EN</option>
      <option value="nb">NB</option>
      <option value="pl">PL</option>
    </select>
  );
}
```
