# Product

## Register

product

## Users

Content creators, social media managers, and marketers who need to produce polished LinkedIn/social media carousel posts quickly. They are visually literate, care about design quality, and will compare the output to Canva, Figma, and Pitch. They use the tool in a focused work session: pick a theme, write content across slides, export, and post.

## Product Purpose

Carouselify lets users create professional carousel posts in minutes, not hours. The core value is eliminating the design bottleneck: pick a template, fill in content, export as PNG/PDF. Success is a user going from idea to posted carousel in under 10 minutes with output that looks like a designer made it.

## Brand Personality

Confident, precise, effortless. The tool should feel like a well-built instrument: every interaction is deliberate, nothing is decorative, the output speaks for itself. Not playful, not corporate. Professional craft.

## Anti-references

- Canva's cluttered, template-browsing, everything-at-once interface
- Generic SaaS admin dashboards with identical card grids
- AI-generated landing pages with gradient text, glassmorphism, and cream backgrounds
- Tools that prioritize features over workflow (too many options, no clear path)

## Design Principles

1. **Output is the product.** The slide canvas is the hero. The editor chrome should feel like a confident instrument around it, not compete with it.
2. **Earned familiarity.** Standard patterns, consistent vocabulary, zero learning curve for anyone who has used a design tool. The tool disappears into the task.
3. **Speed as feature.** Every interaction should feel instant. No loading screens between actions, no confirmation dialogs for low-risk operations, no friction before export.
4. **Confident restraint.** One accent color used with purpose. Clean surfaces. No decorative elements that don't serve the workflow.

## Accessibility & Inclusion

WCAG AA (4.5:1 for text, 3:1 for large text). Keyboard navigation for primary actions. Focus indicators on all interactive elements. Reduced motion support. The tool should work for users with motor impairments (keyboard-only) and low vision (contrast).
